### O que aprendi

<br>

- Aplicar um efeito sobre a grayscale e a saturação de uma imagem:

````css

.grid img {
    width: 100%;
    filter: grayscale(100%);
}

.grid img:hover {
    filter: saturate(105%);
    transform: scale(105%);
    transition: all ease 1s;
}

````
